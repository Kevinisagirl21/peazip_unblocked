PeaZip is a free interactive / batch file archiving / compression utility that provides an unified portable GUI for many Open Source technologies like 7-Zip, FreeArc, PAQ, UPX... free alternative to WinRar, WinZip and similar proprietary software.
- Create 7Z, ARC, BZ2, GZ, *PAQ, PEA, QUAD/BALZ, TAR, UPX, WIM, XZ, ZIP files
- Open and extract ACE, ARJ, CAB, DMG, ISO, LHA, RAR, UDF, ZIPX files and more, over 200 archive types supported
Features of PeaZip includes: archives opener and decompressor, interactive and batch creation and decompression of multiple archives at once, convert files, create sfx archives, split & join files, strong encryption with two factor authentication, encrypted password manager, secure delete, find duplicates, calculate hash / checksum, scripting and scheduling exported job definitions.

PeaZip screenshots: file compression, decompression

Interactive file compression and decompression


First screenshots shows GUI for archive creation and file conversion (extraction and re-compression to new archive format), allowing to compress data to a single output archive or, optionally, compress separately each item in input list to create separate archive files at once.
Archiving GUI also features archive spanning (file split) option, in order to create multiple output volumes of maximum user-defined size.
Same screen allows to add and update files in existing archives for read & write supported formats such as 7z, arc, tar, wim, and zip archives.


Second screenshot displays archive extraction (decompression) GUI, allowing:
  • extraction of multiple archives at once, of same or mixed types, for all full supported or read-supported formats (like ace, rar, zipx etc), decompressing each entire archive
  • extraction of selected files decompressing selected content from a single archive of any read-supported format
In first case the input list contains the archives going to be decompressed, in the second case it will list the files and folders selected for extraction from the archives.

Both archiving and extraction screens features the option to delete original files after compression / decompression operation terminates, optionally using built-in secure data deletion feature.
Password (padlock icon) allows to create / extract encrypted archives.



Batch compression and decompression, scripting & scheduling


Console and Schedule tabs aims to bridge the gap from GUI to CL applications allowing exporting compression / decompression jobs defined in the interactive GUI (scripting) and, on Windows system, scheduling the execution to programmatically automate saved batch jobs.

Console tab is meant to support scripting with ease, automatically generating compression / decompression commands as defined in the interactive GUI and allowing to save and edit the task definition (in example for re-using the scripted task, or for syntax learning purpose) to batch .bat file or for any other type of scripts - UTF-8 encoding is supported.


Schedule tab allows scheduling saved task's execution (i.e. schedule backup and restore jobs) running the script file using system's scheduler to programmatically execute file compression and decompression tasks.
