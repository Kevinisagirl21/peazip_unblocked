PeaZip is a free cross-platform file archiver program that provides an unified portable GUI for many Open Source technologies like 7-Zip, FreeArc, PAQ, UPX... free alternative to WinRar, WinZip and similar proprietary software.
- Create 7Z, ARC, BZ2, GZ, *PAQ, PEA, QUAD/BALZ, TAR, UPX, WIM, XZ, ZIP files
- Open and extract ACE, ARJ, CAB, DMG, ISO, LHA, RAR, UDF, ZIPX files and more, over 200 archive types supported
Features of PeaZip includes: archives opener and extractor, batch creation and extraction of multiple archives at once, convert files, create self-extracting archives, split and join files, strong encryption with two factor authentication, encrypted password manager, secure deletion, find duplicate files, calculate hash, checksum, export job definition as script for programmatical use.

Support PeaZip project development

PeaZip started in 2006 as a spare-time project for a cross-platform, Open Source frontend for various command line compression and file management technologies.

You can donate to PeaZip from paypal.me/peazip

The project employs various third parts technologies, and I'll be glad if you consider about donation to respective Authors: 7-Zip / p7zip, PAQ, FreeArc, and others are all great Open Source projects and deserves all possible support to keep up with development of their excellent programs.
But in first and foremost place a big thank you goes to Lazarus and FreePascal team, providing the cross-platform IDE that makes programming PeaZip possible, easy to test and maintain, and makes possible to port it to many different hardware and software platforms.
