What are WIM, SWM files, Microsoft Windows Imaging Format


Microsoft Windows Imaging Format (WIM) is a bootable file-based disk image format developed by Microsoft for superseding CAB package format in software components / updates distribution on MS Windows platform, from Windows Vista onward, marked by .wim extension.
SWM extension is conventionally used for spanned Windows Imaging Format files. WIM format uses fast LZX compression, a LZ77 family algorithm.

As analogy of scope, not file format features or specifications, WIM for modern Windows platforms has a similar role to TAR on Unix/Linux platforms.

PeaZip fully support working with Windows Imaging Format format both for writing (create, update) and reading (view, browser, extract), both on Microsoft Windows and GNU / Linux operating systems, unlike most other image formats which are read-only supported, like Apple DMG, ISO, UDF, and VMDK formats


Read more: create, open, and extract WIM files, open, and extract CAB files, open and extract DMG ISO UDF files.

External online resources abou .wim archives: WIM Windows Imaging file format specs on Microsoft Technet, LZX compression algorithm on Wikipedia.

SPEED

WIM format features usually high creation / extraction speed, due fast and lightweight LZX compression algorithm.






COMPRESSION RATIO

WIM features usually moderate compression ratio, for most input data types, as LZX algorithm is employed, making it suitable for compression or backup of large pools of data.
ADVANCED OPTIONS N/A
WIM format is usually employed as distribution format for software packages, cannot be evaluated in terms of pure archiving options.
