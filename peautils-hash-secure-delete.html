What is PeaUtils


PeaUtils is a suite of file management utilities, spin-off from PeaZip (file manager and archiver utility), meant for providing a minimal, easy to use GUI to access a collection of most common file management functions:

  • list files and folders (analyzing folder's content) and save list to text file

  • calculate file checksum / hash, including CRC32, CRC64, MD5, SHA1, SHA256, SHA-3_256

  • compare two files byte to byte

  • securely delete files and free space

  • split files and merge spanned volumes

  • hex preview of file
  • display system's environment variables
All packages contains only free, Open Source software under OSI definition, and it does not contain any third party sponsored offers.
PeaUtils is freeware and it is released under LGPLv3 license, free of charge for any use (personal, professional, business, government...), modification, and distribution.
Portable version, not requiring installation and not modifying host system, is availablefor both for Windows and Linux platforms, and it is the easiest way to explore the set of features of this file management toolbox without installation. It provides all functions featured in the installable version: file split, hashing, checksum, secure deletion etc, but it does not, logically, provide out of the box system's context menu integration.

File management utilities guide

Open PeaUtils, clicking on program's shourtcut or on pea.exe file in program's folder

Drag one of more files or folders on it, or use "select files" to open standard system dialog, or click "Search" to start standard system search dialog - then simply drag and drop search matches on the application.

Select one of the file management functions provided by this collection of tools from "Function" dropdown menu. Usually, file tools works on multiple files and folders, exceptions are:
  • File split, File join (merge), and Hex preview, accepts only a single file as input
  • Compare two files byte to byte, accepts one or two files as input; in first case the procedure will ask for selecting the second file from a standard dialog
  • Securely delete free space, and Zero delete free space, does not accept files and folders input. When those functions are selected, the program displays a list of available disk units to perform the required operation on (Windows only)

  • Display environment string does not need input

Alternatively system's context menu (only for Windows installable versions) can be used to pass a single file or folder as input to the program, providing quick access to most common functions of the suite - currently: CRC32 and CRC64 checksum, MD5, SHA1, SHA256, SHA3-256 hash functions, analyze files and folders (detailed listing), compare files byte to byte, hexadecimal preview, display environment variables, secure detete, split and join files.

Once the program is invoked, is possible to add other input, as previously explained.

Context submenu is featured as group on Windows 7 and newer systems (option is available during software's installation).




How to...


Change program's language
PeaUtils is not localized, the GUI and the utilities it collect are only available in English at the moment.

Run as administrator (Windows)
 Option to run the application as administrator should be available rightclicking PeaUtils executable. Alternatively click the hamburger menu (top right corner of the application), and select "Run as administrator". In this way the program will be able to access and securely delete files and folders from reserved paths, which otherwise would end in error due to missing writing credentials.
Updates, help and other online resources
Click the hamburger menu on the top right corner of PeaUtils for links to online resources (help, downloads...).
F1 help hotkey brings to online support page.
